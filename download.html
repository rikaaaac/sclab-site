Release Notes

Version 0.1.0 January 2026

Initial Release

  • Complete single-cell analysis pipeline
  • Support for Cellranger outputs, h5, and h5ad files
  • Preprocessing pipeline: quality control, outlier detection, normalization, feature selection
  • PCA and UMAP/t-SNE dimensionality reduction
  • Leiden clustering algorithm
  • Label cluster groups, Subset cells by metadata, Subset cells by gene expression, annotate cells
  • Differential expression analysis, Co-expression analysis
  • Dataset manager, view analysis history, export results
  • Interactive visualizations of gene expression, cluster and cell groups, and quality metrics
