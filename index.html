What is scLAB?

scLAB is a desktop application designed to make single-cell RNA sequencing analysis accessible to biologists without coding knowledge. Powered by Scanpy, scLAB provides a complete workflow for analyzing single-cell data.

Complete Analysis Pipeline

Perform quality control, normalization, dimensionality reduction, clustering, and differential expression analysis all in one place.

Interactive Visualizations

Explore interactive gene expression plots, clusters plots, and more.

Desktop Application

Fast, secure, and runs entirely on your machine. Your data stays private and analysis runs at native speed.

Powered by Scanpy

Built on top of Scanpy, one of the most used Python libraries for single-cell analysis, ensuring scientific accuracy and reproducibility.

Flexible Data Import

Load outputs from Cellranger pipeline in .h5 format, upload multiple .h5 to merge together different samples or load an already processed .h5ad file.

No Coding Required

All analysis steps are accessible through intuitive point-and-click interactions.
